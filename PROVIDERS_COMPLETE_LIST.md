## ğŸ” API Configuration

All providers require API keys (except local ones). Users configure in Settings page:
- **Required Fields:** API Key, sometimes baseUrl
- **Storage:** LocalStorage only (BYOK - Bring Your Own Keys)
- **Health Check:** Built-in verification per provider
- **Fallback:** If provider fails, app tries next configured provider

## ğŸš€ How to Add New Provider

1. Add to appropriate category in `SettingsPage.tsx`
2. Add to `types.ts` (LLMProviderId, ImageProviderId, etc.)
3. Add to `geminiService.ts` handler method
4. Add UI metadata in `PROVIDER_META`
5. Get API key URL for help link
